### 继承和多态

和JS类似,只能单继承,一个类只能继承一个类,但可以被多个类继承
多态的概念是因为，类的方法可能被重写，表现为不确定性

```php
class Parent {
     public $name; // 可以被继承
     private $age; // 不能被继承
     protected $money; //可以被继承但是不能外部访问
     function __construct($name) {
         $this -> name = $name;
     }
     public function give () {
         echo "give";
     }
 }

$p = new Parent('')

class Child extends Parent {
    function __construct($name) {
        //  $this -> name = $name;
         parent::__construct($name);
     }

    // overwrite 重写方法
     public function give ($give) {

         // 重载父类的方法 , 必须要传参数
         parent::give();
         echo "Child give";
     }

}

```
