#### 理解程序的控制流

程序为实现业务目标所要行进的路径被称为控制流。命令式程序需要通过暴露所有的必要步骤才能极其详细地描述其控制流。


#### 函数链

+ lambda表达式

lambda表达式适用于函数式的函数定义，因为它总是需要返回一个值。

+ R.map()

```javascript
export default function _map(fn, functor) {
  var idx = 0;
  var len = functor.length;
  var result = Array(len);
  while (idx < len) {
    result[idx] = fn(functor[idx]);
    idx += 1;
  }
  return result;
}

```
