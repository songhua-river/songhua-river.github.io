统计一个数字在排序好的数组中出现的位置

### 循环

```javascript
var find = function (arr, target) {
    var i = 0,
        len = arr.length,
        count = 0;
    for (; i < len; i++) {
        if (arr[i] === target) {
            count++;
        }
    }
    return count;
}
```

因为式排序数组所以找到后可以跳出

```javascript
var find = function (arr, target) {
    var start,
        i,
        len = arr.length;
    for (i = 0; i < len; i++) {
        if (arr[i] === target && start === undefined) {
            start = i;
        } else if (arr[i] !== target && start !== undefined) {
            return i - start;
        }
    }
    return 0;
}
```
