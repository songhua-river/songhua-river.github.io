```javascript
    function foo(){
        console.log(length);
    }
    function bar(){
        var length ='杭州';
        foo();
    }
    bar()

```

#### 答案

```javascript
    //0
```
#### 解析

+ 首次运行,执行foo函数在其作用域内寻找length,发现没有定义

+ 并不会查找bar作用域内的length,函数的作用域是在函数定义时创建的，函数执行后作用域随之销毁，所以向上寻找到的应该是全局的作用域。

+ 全局作用域中的length表示的页面中iframe的数量，所以window.length === 0;
